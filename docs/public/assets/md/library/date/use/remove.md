
# Remove

Removes time unit to the OnesyDate value.

```ts
const onesyDate = new OnesyDate();

remove(14, 'day', onesyDate);
```

### Units

- millisecond
- milliseconds
- second
- minute
- hour
- day
- dayWeek
- dayYear
- week
- month
- year

## API

#### remove

```ts
default function remove(value: number, unit: TTimeUnits, onesyDate?: OnesyDate): OnesyDate;
```
